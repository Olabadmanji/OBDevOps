SmartReceipt – Automated Receipt Processing System

SmartReceipt is a serverless, event-driven receipt processing pipeline built on AWS. The system automatically extracts and structures receipt data when files are uploaded to Amazon S3.

Project Objective

To design a scalable backend automation system that simulates real-world financial document processing using AWS serverless architecture.

Architecture Overview

User Upload → Amazon S3 (Private Bucket) → AWS Lambda → Amazon Textract → Amazon DynamoDB → CloudWatch Logs
  • Receipt uploaded to private S3 bucket
  • S3 event triggers Lambda automatically
  • Lambda sends document to Textract for OCR
  • Extracted data parsed and structured
  • Structured data stored in DynamoDB
  • System logs recorded in CloudWatch

AWS Services Used

  • Amazon S3 – Secure file storage
  • AWS Lambda – Serverless processing
  • Amazon Textract – OCR extraction
  • Amazon DynamoDB – Structured data storage
  • Amazon CloudWatch – Monitoring & logging
  • IAM – Secure access control

Security Implementation

  • S3 public access disabled
  • Least privilege IAM roles
  • No hardcoded credentials
  • Private backend architecture

Technical Highlights

  • Event-driven serverless design
  • Real-time processing pipeline
  • OCR-based data extraction
  • Structured NoSQL storage
  • Cloud-native monitoring

How It Was Tested

  • Manual upload of PDF/JPG receipt files to S3
  • Verified Lambda execution in CloudWatch
  • Confirmed extracted data stored in DynamoDB
  • Validated OCR response from Textract

Future Improvements
