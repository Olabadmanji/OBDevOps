Event Announcement System

A serverless event notification system built with AWS services.

Overview

This project is a cloud-based Event Announcement System that allows event organizers to publish announcements and notify subscribers via email in real time.

It demonstrates serverless architecture using AWS Lambda and SNS, with a static frontend hosted on Amazon S3.

Architecture

Tech Stack

  • HTML, CSS, JavaScript
  • AWS S3 (Static Website Hosting)
  • AWS Lambda (Serverless backend)
  • AWS SNS (Email notifications)
  • IAM (Permissions & security)
  • GitHub (Version control)

Key Features

  • Static frontend hosted on Amazon S3
  • Serverless backend using AWS Lambda
  • Email notifications via Amazon SNS
  • Scalable and cost-efficient architecture
