Event Announcement System – Serverless Event Notification System

A serverless event notification system built with AWS services.

Project Objective

To build a cloud-based Event Announcement System that allows event organizers to publish announcements and notify subscribers via email in real time using a serverless AWS architecture.

Architecture Overview

Static Frontend (Amazon S3) → AWS Lambda → Amazon SNS → Email Subscribers
  • Static frontend hosted on Amazon S3
  • Serverless backend using AWS Lambda
  • Email notifications via Amazon SNS
  • Scalable and cost-efficient architecture

Tech Stack

  • HTML, CSS, JavaScript
  • AWS S3 (Static Website Hosting)
  • AWS Lambda (Serverless backend)
  • AWS SNS (Email notifications)
  • IAM (Permissions & security)
  • GitHub (Version control)
